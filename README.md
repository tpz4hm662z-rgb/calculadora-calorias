# Calculadora de Calorías | Herramientas360

Calculadora gratuita para estimar las calorías diarias necesarias según datos personales y nivel de actividad física.

## Características

- Cálculo de calorías de mantenimiento.
- Estimación para perder peso.
- Estimación para ganar peso.
- Interfaz responsive.
- Diseño profesional basado en Herramientas360 Template v3.0.
- Optimizada para SEO.
- Compatible con móviles, tablets y ordenador.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
